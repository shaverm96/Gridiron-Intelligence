Workflow Description:

  • Tier 1 - Routing: Player Resolution Engine attempts to match user query to player in database
  • Tier 2 - Data Retrieval: If found, retrieves data from multiple sources (Master Index, Recruit Profile, Stats, ML Model)
  • Tier 2b - Gap Filling: If player not found or data incomplete, searches web archives and news
  • Tier 3 - Inference: Runs live XGBoost predictions and generates visual tier charts
  • Tier 4 - Synthesis: Combines all data into context and uses LLM to generate final scout report
